    Training preparation

    Training preparation ini terdiri dri 3 step

    1. model

    model merupakan arsitektur yg kita udh buat tadi, tgal assignment aja ke variabel model







    2. Criterion

    Criterion merupakan sebuah fungsi yg dipakai untuk menghitung loss


    - kalo diklasifikasi kita pakai MSE gradient descentnya gagal karena ada platou, sehingga kita mau menghindari itu sehingga diklasifikasi kita menggunakan yg nama MCE(mean cross entrophy)

    - MCE(mean cross entrophy) sebenarnya formula matematiknya adalah logsoftmax dikombinasikan NLL(negatif log likelihood)


    - jadi Criterion kan digunain buat ngehitung erro/loss yaa nah ini digunakan tergantung kasusnya apa in this case kita make sigmoid 0 1 (binary clf)

    - kalo kasusnya multiclass dan aktivasi terkahir kita pake logsoftmax Criterionnya kita pake NLLLoss gt aja pkonya

    - kalo kasusnya kita regresi yaa kita pake nn.MSELoss jadi tergantung bgt sama kasus kita


    AdamW = Adaptive Momentum + weight decay, penjelasanya dibawah

    inputnya dia minta apa?? dia minta tolong isi semua parameter neural network kita

    sehingga kita bisa menggunakan (model.parameters) lr = bisa diatur tp kita pake default aja



    - oke jadi dideep learning kita gaakan pake gradient descent biasa untuk trainingnya karena gradient descent rentan terjebak diplatou addle point dkk

    - sehingga di deep learning kita tidak pake optimizer basic lg tp kita akan memanfaatkan ide baru

    - ideanya namanya adalah momentum, apa itu momentum??

    - bayangin aja kaya jatohin bola dia itukan ada momentumnya kan yaa walaupun udh didasar dia akan kelewatan bisa jadi turun lg

    - artinya apa?? kalo ada local minima dkk harapnya tembus dan bisa menemukan daerah yg dibawahnya lagi

    - lagi lagi gaada jaminan kita dapet global minimum, dan kita sebenarnya gaperlu perlu bgt dapet global minimum, yg penting dpet yg lebih rendh jangan yg diatas atas

    - tp kita gabakal pake momentum biasa kita akan pakai momentum dari momentum = Adaptive Momentum(Adam)

    - tapi kita akan pakai adam biasa tp kita akan pakai yg namanya AdamW = Adaptive Momentum + weight decay

    - bedanya apa AdamW ada tambahan L2 regularization jadi bisa mengurangi overfitt
